toki pona keyboard layout


I recently created a keyboard layout for toki pona. Now, you might be like "why ?". Well, I have 3 reasons for creating this beauuutiful layout:

  • the Qwerty keyboard (or any other keyboard layout) has a lot of letters that are not used in toki pona
  • these keyboard layouts don't have the most used letters in toki pona in their center
  • there can be a kijetesantaklu key

So, with the helpful collaboration of @skrybl, I started creating the keyboard layout. The first thing I did was scan a long text in toki pona I had written to see the most commonly used letters.
